# kubeapt policies download

Download policies into local storage. Optionally specify a version.

## Usage

```bash
kubeapt policies download [flags]
```

## Subcommands

None.

## Arguments

| Flag | Type | Default | Description |
| --- | --- | --- | --- |
| --version | string |  | Policy version to download (defaults to latest) |
| --log-level | string | info | Set logging level (debug, info, warn, error) |
| --kubeconfig | string |  | Path to kubeconfig file |
| -h, --help | bool | false | Show help for this command |
