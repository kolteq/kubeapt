# kubeapt policies remove

Remove a downloaded policy version from local storage. Optionally specify a version.

## Usage

```bash
kubeapt policies remove [flags]
```

## Subcommands

None.

## Arguments

| Flag | Type | Default | Description |
| --- | --- | --- | --- |
| --version | string |  | Policy version to remove (defaults to latest downloaded) |
| --log-level | string | info | Set logging level (debug, info, warn, error) |
| --kubeconfig | string |  | Path to kubeconfig file |
| -h, --help | bool | false | Show help for this command |
