# kubeapt policies list

List available policy versions. Use --local to show only downloaded versions.

## Usage

```bash
kubeapt policies list [flags]
```

## Subcommands

None.

## Arguments

| Flag | Type | Default | Description |
| --- | --- | --- | --- |
| --local | bool | false | List only locally downloaded policy versions |
| --log-level | string | info | Set logging level (debug, info, warn, error) |
| --kubeconfig | string |  | Path to kubeconfig file |
| -h, --help | bool | false | Show help for this command |
