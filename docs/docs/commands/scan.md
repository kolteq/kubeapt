# kubeapt scan

Scan the connected cluster for admission safeguards, webhooks, and policy updates. Run it against the current kubeconfig context.

## Usage

```bash
kubeapt scan [flags]
```

## Subcommands

None.

## Arguments

| Flag | Type | Default | Description |
| --- | --- | --- | --- |
| --log-level | string | info | Set logging level (debug, info, warn, error) |
| --kubeconfig | string |  | Path to kubeconfig file |
| -h, --help | bool | false | Show help for this command |
