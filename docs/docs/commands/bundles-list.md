# kubeapt bundles list

List available policy bundles and versions. Use --local to show only downloaded bundles.

## Usage

```bash
kubeapt bundles list [flags]
```

## Subcommands

None.

## Arguments

| Flag | Type | Default | Description |
| --- | --- | --- | --- |
| --local | bool | false | List only locally downloaded bundles |
| --log-level | string | info | Set logging level (debug, info, warn, error) |
| --kubeconfig | string |  | Path to kubeconfig file |
| -h, --help | bool | false | Show help for this command |
