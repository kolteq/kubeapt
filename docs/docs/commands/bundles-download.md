# kubeapt bundles download

Download a policy bundle into local storage. Provide the bundle name as the required argument.

## Usage

```bash
kubeapt bundles download <bundle-name> [flags]
```

## Subcommands

None.

## Arguments

| Flag | Type | Default | Description |
| --- | --- | --- | --- |
| --version | string |  | Bundle version to download (defaults to latest) |
| --log-level | string | info | Set logging level (debug, info, warn, error) |
| --kubeconfig | string |  | Path to kubeconfig file |
| -h, --help | bool | false | Show help for this command |
